## Aplicación profesional

Durante este módulo comprendí cómo los servicios de red de Azure permiten extender una infraestructura corporativa hacia la nube manteniendo principios de segmentación, seguridad y alta disponibilidad.

Pude relacionar Azure Virtual Network y las subredes con el diseño de redes empresariales tradicionales, entendiendo que la segmentación facilita el aislamiento de servicios, la aplicación de controles de seguridad y la reducción de riesgos operacionales.

También identifiqué las diferencias entre Azure VPN Gateway y Azure ExpressRoute. Mientras una VPN representa una alternativa segura y de menor costo para conectar infraestructuras, ExpressRoute ofrece una conexión privada, con menor latencia, mayor disponibilidad y mayor previsibilidad, siendo especialmente apropiada para organizaciones con cargas críticas o altos requerimientos de seguridad y continuidad operacional.

Finalmente, comprendí la importancia de Azure DNS como componente esencial para la resolución de nombres y el acceso a los servicios, reforzando la idea de que una arquitectura cloud no depende únicamente de los recursos de cómputo, sino también de una correcta planificación de la conectividad y de los servicios de red.

Este módulo reforzó conceptos que ya conocía por mi experiencia en infraestructura y telecomunicaciones, permitiéndome comprender cómo esos mismos principios se implementan y gestionan dentro del ecosistema de Microsoft Azure.
