## Aplicación profesional

Durante este módulo identifiqué cómo los distintos servicios de proceso de Azure pueden resolver necesidades reales en entornos empresariales.

Pude relacionar Azure Virtual Machines con escenarios donde se requiere un control completo del sistema operativo, como la instalación de aplicaciones heredadas o configuraciones específicas.

Comprendí el valor de los contenedores para garantizar la consistencia entre los entornos de desarrollo, pruebas y producción.

Además, identifiqué un caso de uso concreto para Azure Functions, automatizando el aprovisionamiento de usuarios en Microsoft Intune mediante reglas de negocio basadas en el país o la organización, aplicando automáticamente grupos, configuraciones y políticas.

Este ejercicio me permitió comenzar a pensar en los servicios de Azure no solo desde una perspectiva técnica, sino también como herramientas para resolver necesidades reales del negocio mediante automatización, escalabilidad y eficiencia operativa.
